Specific View Configuration

Specific View Configuration is a group of the option items related to specific views. This category includes the [UseCase View], [Actor View], [Enumeration View], [Interface View], [Artifact View], [Component View / Component Instance View] and [Node View / Node Instance View] subcategories.

[UseCase View] Option

Option Item

Default

Description

Show stereotype

Text

Specifies the default stereotype indication method for UseCase view elements (Text, Icon, None, Decoration or hide).

Suppress attribute

True

Specifies whether to suppress the attributes for UseCase view elements.

Suppress operation

True

Specifies whether to suppress the operations for UseCase view elements.

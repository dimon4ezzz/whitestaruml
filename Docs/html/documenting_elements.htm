Documenting Elements

Documentation is very important for capturing thoughts, development notes, description of the element and other information. The documentation of an element can be used when generating paper documentation, this way you will automatically have the diagrams and the notes together.

To add documentation to a model element:

1.In the Model Explorer or the diagram, select an element.
2.In the inspector area, open the Documentation tab.
3.Enter your documentation in the text box.

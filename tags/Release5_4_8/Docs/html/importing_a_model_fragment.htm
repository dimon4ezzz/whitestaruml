Importing Model Fragments

You can import elements saved in a model fragment file into your project. Importing a model fragment copies the elements contained in the model fragment into the project. Your project will not store a reference to the imported fragment file.

To import a model fragment

1.On the File menu, point to Import, and then click Model Fragment.
2.In the Open Model Fragment dialog box, select a model fragment file and click Open.

3.In the Select a Package where the Model Fragment will be imported to dialog, choose a package and click OK.

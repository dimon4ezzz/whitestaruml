Adding Attachments to Elements

Attach related files or web page URLs to elements for extra documentation. You can easily access the attached files or web pages from  within WhiteStarUml.

To attach a file

1.In the Model Explorer, select an element.
2.In the inspector area, open the Attachments tab.

Attachments

3.Click .
4.In the Attachment dialog, enter the full pathname and filename of the attachment file or the web page URL (or click the browse button on the right to select from the browse window), and click the [OK] button.
